# 🛒 Smart Grocery Delivery System

## 📌 Overview

A C++ project that simulates an online grocery store with customer management, inventory handling, and delivery routing using data structures and algorithms.

## ⚙️ Features

* Customer data using Binary Search Tree (BST)
* Grocery items stored using Hash Table (O(1) access)
* Shortest path delivery using Dijkstra’s Algorithm
* Insert, delete, update, and search operations

## 🛠️ Tech Stack

* C++
* BST, Hash Table, Graphs
* Dijkstra Algorithm
